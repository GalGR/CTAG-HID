# C-TAG HID

## Prerequisites
```pip3 install hid```

## Installation
Your installtion folder should look like this:
```
.
├── install_folder
│   ├── ctag-hid.py
│   ├── include_dll_path
│   │   └── __init__.py
│   ├── x64
│   │   └── hidapi.dll
│   └── x86
│       └── hidapi.dll
```
